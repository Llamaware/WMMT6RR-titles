소개 / Summary

일일칭호는 보유차종 선택 후 표시되는 차종정보 페이지를 넘긴 횟수에 따라 획득할 수 있는 칭호이다.

일일칭호의 공통출현조건은 다음과 같다.

  • 1번 칭호 : 차종정보 화면에서 뷰 체인지(View change) 버튼 (초록색 버튼)을 9~17회 눌러 차량정보 페이지 전체를 1회 전환
  • 2번 칭호 : 차종정보 화면에서 뷰 체인지 버튼을 18~26회 눌러 차량정보 페이지 전체를 2회 전환
  • 3번 칭호 : 차종정보 화면에서 뷰 체인지 버튼을 27~35회 눌러 차량정보 페이지 전체를 3회 전환
  • 4번 칭호 : 차종정보 화면에서 뷰 체인지 버튼을 36~44회 눌러 차량정보 페이지 전체를 4회 전환
  • 5번 칭호 : 차종정보 화면에서 뷰 체인지 버튼을 45~53회 눌러 차량정보 페이지 전체를 5회 전환
  • 6번 칭호 : 차종정보 화면에서 뷰 체인지 버튼을 54회 이상 눌러 차량정보 페이지 전체를 6회 이상 전환
